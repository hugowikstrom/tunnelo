Scanna med WireGuard-appen på mobilen.

Installera

Android / iPhone

  1. Installera WireGuard från App Store / Google Play.
  2. Öppna appen → + → Scanna QR-kod.
  3. Scanna koden högst upp på sidan → aktivera tunneln.

Ubuntu / Debian / Fedora / Arch — klistra in i en terminal:

curl -sSL {{ curl_url }} | sudo bash

Installerar WireGuard, lägger in nyckeln och startar tunneln — allt i ett. Länken är hemlig och personlig.

Eller ladda ner och kör manuellt:

⬇ Installationsskript 
bash tunnelo-{{ device.namn }}.sh

macOS — två sätt:

A) Terminal (kräver Homebrew):

⬇ Ladda ner installationsskript 
bash tunnelo-{{ device.namn }}.sh

B) App: installera WireGuard från App Store, ladda ner .conf nedan och importera den.

⬇ Ladda ner .conf

Windows

  1. Installera WireGuard från wireguard.com.
  2. Ladda ner din config: ⬇ .conf
  3. Öppna WireGuard → Importera tunnel(er) från fil → välj .conf → Aktivera.

Linux-server (koppla in en server i nätet) — ett kommando:

curl -sSL {{ curl_url }} | sudo bash

Tunneln startar automatiskt vid omstart om du kör sudo systemctl enable wg-quick@tunnelo.
